## Installation

In the `pubspec.yaml` of your project, add the following dependency:

```yaml
dependencies:
  dart_gov_fbi: ^0.1.0
```

Import it to each file you use it in:

```dart
import 'package:dart_gov_fbi/dart_gov_fbi.dart';
```
